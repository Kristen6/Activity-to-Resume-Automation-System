# Activity-to-Resume Automation System

## Team Members
- Junzhe Zhang, Haowei Zhu, Yang Wang

## Project Description
This project proposes the design and implementation of a lightweight, human-centered system that allows users to seamlessly capture activities as they happen and automatically transform them into structured, high-impact representations suitable for resumes, CVs, or personal webpages.

The system enables users to:
- Quickly record activities with minimal effort (e.g., short text input, voice input, or simple prompts)
- Capture essential context such as date, role, organization, duration, and type of activity
- Automatically organize and categorize activities (e.g., research, leadership, volunteering, creative work)
- Translate raw activity logs into resume-ready bullet points, CV sections, or structured profiles
- Generate or update artifacts such as a resume, CV, or personal webpage with minimal manual editing
